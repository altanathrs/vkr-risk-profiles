# vkr-risk-profiles
Сравнительный анализ риск-профилей поколений Y и Z на российском рынке

# О работе 
Выпускная квалификационная работа посвящена сравнительному анализу риск-профилей инвесторов поколений Y (миллениалы) и Z (зумеры) на российском финансовой рынке на основе авторского опроса. 

# Данные
Метод сбора: авторский онлайн-опрос
Выборка: 182 респондента (89 миллениалов и 93 зумера)
Период сбора: 2026 год
Критерий включения: наличие опыта инвестирования или сбережения 

# Инструментарий
Python-Jupiter Notebook-pandas-numpy-scipy.stats-statsmodels-matplotlib-seaborn-Excel

# Методы анализа 
-t-тест Стьюдента/Уэлча
-Критерий X^2 Пирсона + коэффициент V Крамера 
-Поправка Бонферрони
-Множественная линейная регрессия (OLS)
-d Коэна (размер эффекта)
